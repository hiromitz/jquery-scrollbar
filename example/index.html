Custome (Simple) Scrollbar plugin for jQuery

Add custom scrollbar to elements. Plugin that works with jQuery.
(additionally jQuery Mouse Wheel plugin for mousewheel action.)

Usage

$('#element').scrollbar(options)

Options

name type default description
axis string 'y' horizontal or vertical scroll direction. x | y
wheel number 20 scroll step on mousewheel
animate boolean true animation on scroll
fade boolean false fade in / out scrollbar on hover
prefix string 'jsb-' css class prefix

Methods

.scrollbar('update')

update scrollbar size and position. ( use this method after elements changed )

$('#element').scrollbar('update')
